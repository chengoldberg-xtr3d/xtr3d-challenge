# Xtr3d Challenge
## Requirements

### Game
Python 2.7.x with the following packages:

1. numpy
2. opencv-python version 2.4.9 - 2.4.12
3. pygame

### Notebooks
The following packages:

1. ipython
2. jupyter
3. matplotlib
